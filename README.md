# Zenith

Site web de **Zenith**, une agence de développement et design web sur mesure.
Bilingue (FR/EN), dark/light mode, optimisé SEO.

</div>

---

## Stack technique

- **Framework** : [Nuxt](https://nuxt.com/) v4.2.2
- **Langage** : TypeScript
- **UI** : Tailwind CSS + shadcn-nuxt + Radix Vue
- **Internationalisation** : @nuxtjs/i18n (FR / EN)
- **Contenu** : @nuxt/content (Markdown)
- **Base de données** : SQLite (better-sqlite3)
- **Formulaires** : Vee-validate + Zod
- **Email** : EmailJS + Resend
- **Déploiement** : Vercel

## Prérequis

- [Bun](https://bun.sh/) ≥ 1.2.14
- [Node.js](https://nodejs.org/) ≥ 23

## Scripts disponibles

| Commande | Description |
|----------|-------------|
| `bun run dev` | Lance le serveur de développement |
| `bun run dev:http` | Lance le serveur avec accès réseau (`--host`) |
| `bun run build` | Compile pour la production |
| `bun run generate` | Génère le site statique |
| `bun run preview` | Prévisualise le build de production |
| `bun run lint` | Vérifie le code avec ESLint |
| `bun run lint:fix` | Corrige les erreurs ESLint automatiquement |
| `bun run format` | Formate le code avec Prettier |
| `bun run format:check` | Vérifie le formatage Prettier |
| `bun run typecheck` | Vérifie les types TypeScript |
